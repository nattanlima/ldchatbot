Perguntas Frequentes

Tire suas dúvidas sobre a migração para API Oficial.

Posso usar meu número atual?
Sim! Nós fazemos a portabilidade do seu número atual para a API Oficial. Você não perde o número que seus clientes já conhecem. O processo leva menos de 1 hora.

Perco minhas conversas antigas?
Ao migrar para a API Oficial, o histórico de conversas do aparelho é resetado pela Meta. Porém, recomendamos fazer um backup antes. A partir da migração, todo o histórico fica salvo na nuvem da Prisme para sempre.

Preciso ter CNPJ?
Sim, a Meta exige um CNPJ ativo e um site (ou página social) para verificar sua empresa e conceder o selo de API Oficial. Nossa equipe auxilia em todo esse processo de verificação.

O que é a tecnologia COEX?
COEX (Coexistence) é uma nova tecnologia da Meta que permite usar a API Oficial e o App do WhatsApp Business no celular SIMULTANEAMENTE. Antigamente, ao usar API, você perdia o acesso no celular. Com a Prisme, você tem os dois.
